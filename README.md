# Pi-Compute-in-Java
Fast Pi-Compute Java method using rapid convergence Chudnovsky algorithm
## Performance Benchmarks
Approximate Time to compute π on local machine:
* 2 ms to calculate 100 digits
* 30 ms to calculate 1000 digits
* 1 sec to calculate 10,000 digits
* 4 sec to calculate 20,000 digits
* 40 sec to calculate 50,000 digits
* 216 sec to calculate 100,000 digits
